# Hệ thống cảnh báo nhiệt độ, độ ẩm và nồng độ khí CO

## Nguyên liệu
### Client
- Meteor Platform
- Boostrap
- JQuery
### Server
- NodeJS
- MongoDB

## Hướng dẫn sử dụng
- Bật thiết bị IOT
- Bật MongoDB
- Mở Terminal, nhập vào lần lượt các lệnh
cd server
node 'tcp_server.js'
- Mở thêm 1 Terminal, nhập vào lần lượt các lệnh
cd server
node 'node js server.js'
- Mở thêm 1 Terminal, nhập vào lần lượt các lệnh
cd CHT_KIT
meteor
